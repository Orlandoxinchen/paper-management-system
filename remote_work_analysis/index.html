远程办公数据 — 分析脚本使用说明

从 远程办公数据.xlsx 重算 Table 2～Table 8 与 Figure1，单文件复制即跑。

1. 下载脚本（任选一种）

  • 直接下载 remote_work_full.py（推荐） → 浏览器会打开代码页，右键「另存为」保存为 remote_work_full.py
  • 或打开 仓库里的脚本 → 点 Raw → 另存为

2. 准备数据

把 远程办公数据.xlsx 放在桌面。

例如 Mac 路径：/Users/DoDo/Desktop/远程办公数据.xlsx

脚本会自动在「当前用户桌面」或 /Users/DoDo/Desktop 下查找该文件；结果也会输出到桌面。

3. 安装依赖

打开终端，执行：

pip3 install pandas numpy scipy statsmodels matplotlib openpyxl factor_analyzer

若提示 python: command not found，请用 python3 和 pip3；先运行 python3 --version 确认已安装。

4. 运行脚本

在终端进入脚本所在目录（若在桌面）：

cd ~/Desktop
python3 remote_work_full.py

运行时会先显示「数据文件」「输出目录」；Bootstrap 阶段会提示「约 1000 次，请稍等」，属正常现象。

输出文件（默认在桌面）

  • Table2_reliability_validity.csv
  • Table3_pearson_correlation.csv
  • Table4_chain_mediation_paths.csv
  • Table5_bootstrap_indirect_effect.csv
  • Table6_sem_results.csv
  • Table8_multigroup_male.csv
  • Figure1_Bootstrap_BW.png
